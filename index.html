learnyouaframe

A-Frame is a web framework for building virtual reality experiences. This workshopper aims at introducing this framework to web developers.

Exercises

Exercises are preloaded in a directory with html files for you to edit. You can click on OPEN DIRECTORY to view exercises directory in your OS's default file explorer.

How learnyouaframe works

For each lesson there is an expected and an actual markup. The former is provided by us, and your goal is to simply modify the actual markup so that it matches the expected one. You can see the preview of your markup by clicking on PREVIEW.

Verifying the Lesson

Once you're confident that the two markups are equivalent, you can click the VERIFY button (and hopefully, pass!) the lesson.
